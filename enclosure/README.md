## Design Requirements

The enclosure should accommodate:
- XIAO ESP32-C6 (21mm x 17.5mm)
- R502-A fingerprint sensor (requires ~20mm diameter opening)
- USB-C port access for charging/programming
- Optional: mounting holes for attachment

## Contributing

3. Photos of the printed result
